# Directory Overview

This directory contains study materials for a microbiology course. It includes PDF files and detailed Markdown notes summarizing the content of those presentations.

# Usage

This directory is intended for studying microbiology. The `.pdf` files are the primary course materials, and the `.md` files are supplementary notes that provide a detailed, text-based summary of the presentations. The Markdown files are useful for quick review and searching for specific topics.

# Microbiology Study Materials

This repository contains study materials for Microbiology course (2025).

## Directory Structure

*   ~~`pdf/`: Contains PDF presentations for each topic.~~ Will be hosted privately due to licensing issues.
*   `_summary/`: Contains detailed Markdown notes summarizing the content of the PDF presentations. These summaries were generated using Gemini.
*   `quiz/`: Contains quizzes related to the course material.

## Usage

The `.pdf` files are the primary course materials. The Markdown files in `_summary/` provide a detailed, text-based summary of the presentations, useful for quick review and searching for specific topics. The `quiz/` directory contains quizzes to test your knowledge.
